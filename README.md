# JBower
Pure Java Alternative to Bower


## Bower Command line support
| Command   | Bower     | JBower      |
|-----------|:---------:|:-----------:|
| cache     | supported |             |
| help      | supported |             |
| home      | supported |             |
| info      | supported |             |
| init      | supported |             |
| install   | supported |             |
| link      | supported |             |
| list      | supported |             |
| lookup    | supported |             |
| prune     | supported |             |
| register  | supported |             |
| search    | supported | in progress |
| update    | supported |             |
| uninstall | supported |             |
| version   | supported |             |
